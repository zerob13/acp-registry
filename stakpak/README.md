# Stakpak

Open-source DevOps agent in Rust with enterprise-grade security.

## Features

- **Security**: mTLS encryption, dynamic secret redaction, privacy mode
- **Task Management**: Asynchronous task execution with real-time progress tracking
- **Infrastructure as Code**: Generate Terraform, Kubernetes, Dockerfile, and GitHub Actions
- **Code Analysis**: Semantic understanding and documentation analysis

## Getting Started

1. Install from the ACP registry
2. Get a free API key from [stakpak.dev](https://stakpak.dev)
3. Set the `STAKPAK_API_KEY` environment variable

## Platform Support

- macOS (Apple Silicon and Intel)
- Linux (x86_64)
- Windows (x86_64)
